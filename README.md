# Tic Tac Toe in Godot

<hr>

Entire Tic Tac Toe game example in godot 3.1 or up game engine.

## Features
 
- [x] Player vs Player
- [ ] Player vs Ai
- [ ] Player vs Player (in Local Network)
- [ ] Score counting system
- [ ] Start Screen
